# Card Sweeper

A card matching game


## Info :dart:
- Match cards
- Matching two skulls will end the game
- Matching two crosses, a skull will be removed from the board

<hr>

## Controls :joystick: 

|Key|Action|
|:---:|:---:|
|Left Mouse|Flip Over Card|
